# 🏀 Sportory

**Not Just a Game. It’s Your Story.**

Sportory is a sports platform built to empower Ghanaian athletes and fans—without betting. It’s a place to showcase talent, build community, and celebrate the spirit of sport.

---

## 🚀 MVP Features

### ✅ Core Modules
- **User Authentication** – Email/phone login with role-based access (Athlete, Fan, Coach, Admin)
- **Athlete Profiles** – Upload bio, sport, stats, and highlight videos
- **Fan Challenges** – Weekly prediction games (non-monetary) with leaderboards
- **Community Feed** – Posts, likes, comments, and admin moderation
- **Merch Marketplace** – Product listings, cart, and mobile money checkout (placeholder)
- **Match Streaming** – _Coming Soon!_ 🎥 We're setting up the gear to bring you live action.

---

## 🛠 Tech Stack

| Layer       | Tech                     |
|------------|--------------------------|
| Frontend   | React + Tailwind CSS     |
| Backend    | PHP (FastRoute) or Node.js |
| Auth       | Firebase Auth or Supabase |
| Storage    | Supabase Buckets or Firebase Storage |

## 📦 Setup Instructions

```bash

cd sportory

# Install dependencies
npm install

# Start development server
npm run dev
```

---

## 🔧 Deployment Notes

Ensure `vercel.json` is configured for SPA routing:

```json
{
  "rewrites": [
    { "source": "/(.*)", "destination": "/" }
  ]
}
```

Push to GitHub and deploy via Vercel.
